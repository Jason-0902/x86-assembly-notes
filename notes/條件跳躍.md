# 條件跳躍

## 等於 / 不等於

| 指令 | 條件 |
| --- | --- |
| je | ZF = 1 |
| jne | ZF = 0 |

## Signed 比較

| 指令 | 意義 |
| --- | --- |
| jg | > |
| jl | < |

## Unsigned 比較

| 指令 | 意義 |
| --- | --- |
| ja | > |
| jb | < |

## Signed vs Unsigned

```python
mov rdi, -1
```

在 signed:

```python
-1
```

所以:

```nasm
cmp rdi, 1
jg  ; 不跳
ja  ; 會跳
```

這就是很多條件繞過漏洞的來源
