# 基本指令

## `mov`

```
mov rax, rdi
```

代表:

```
rax = rdi
```

## `add`

```
add rax, rsi
```

代表:

```
rax = rax + rsi
```

Assembly 是**破壞式運算**

## `imul`

```
imul rax, 2
```

代表：

```
rax = rax * 2
```

---

## `Shift（位元思維）`

```
shl rax, 1  → ×2
shl rax, 2  → ×4
shl rax, 3  → ×8
```

因為:

```
左移 n bit = × 2^n
```

## `LEA`

```
lea rax, [rdi + rdi*4]
```

代表:

```
rax = 5 * rdi
```

LEA 不只是算地址

Compiler 常用它當免費乘法器
