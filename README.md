# x86-assembly-notes

My structured study notes on x86-64 assembly, low-level internals, reverse engineering, and binary exploitation.

This repository documents my journey into understanding how software really works under the hood.

---

## What This Repo Covers

- x86-64 registers and calling conventions
- Stack frame internals
- Memory layout (stack, heap, .text, .data)
- Function prologue & epilogue
- Instruction-level behavior
- Reverse engineering techniques
- Exploit development fundamentals

---

## Why I'm Learning This

Understanding assembly is critical for:

- Reverse engineering
- Vulnerability research
- Binary exploitation
- Fuzzing
- Operating system internals
- Security research

High-level languages hide implementation details — assembly reveals them.
